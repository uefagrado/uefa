Recursos de trabajo



Esta es la página de descargas del sitio web, acá estaran disponibles los entornos para trabajar en Visual Basic que ya estén obsoletos o no séan faciles de hallar. Entre estos entornos estarán el Visual Basic 6.0 que nos permiten trabajar exclusivamente Visual Basic y los entornos Visual Studio, 2005, 2010 y 2012 que permiten trabajar Visual Basic como una opción ademas de poseer una interfaz mas pulida, manejable y estética.

Visual Basic 6.0

Entorno base para trabajar Visual Basic 6, versión de 64 bits.

Visual Studio 2005

Visual Studio 2005 de 32 Bits, en la descarga esta un texto con el serial. Permite trabajar Visual Basic, C++ y C#.

Visual Studio 2008

Visual Studio 2008, ya activado. Permite trabajar Visual Basic, C++,C# y Visual Basic.Net.

Visual Studio 2010

Esta versión de Visual Studio es incompleta y solo tiene la función de trabajar en Visual Basic.

Visual Studio 2012 Ultimate

Visual Studio 2012 con todas sus características, descarga directa para 32 bits. El serial del programa es : YKCW6-BPFPF-BT8C9-7DCTH-QXGWC
