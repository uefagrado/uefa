Lenguaje de programación Visual Basic 6.0


Visual Basic, es un lenguaje de programación que sirve como variante de BASIC y además está orientado a eventos.

Visual Basic se caracteriza por el hecho de que posee una interfaz de desarrollo que nos brinda la posibilidad de poder ubicar elementos de entrada, salida y acción a nuestro programa sin programar de antemano, es decir, que los elementos estan creados para solo tocar y arrastrar. Visual Basic funciona haciendo uso de formularios(lo que sería nuestra ventana de programa) donde pondremos todos los elementos que necesitemos, un área de programación donde le daremos funcionalidad a los elementos de nuestro programa y un área de propiedades donde podremos cambiar aspectos como tamaño, grosor, colores y estilos de los elementos que usamos, aquí un ejemplo sencillo de una ventana en Visual Basic:

Private Sub Form_Load()
MsgBox ("¡Hello, world!")
End Sub

Ventajas de Visual Basic


Se puede encontrar de forma nativa(predefinida) en algunas aplicaciones de la suite ofimática Office, tales como en Word, Powerpoint, Excel y Access.

Resulta fácil de aprender para aquellos que apenas inician en la programación.

La programación de las funciones de cada elemento resulta sencilla al venir la mayoría ya predefinidas,



Desventajas de Visual Basic


Al venir la mayoría de eventos, funciones y objetos ya creados, a pesar de tener grandes horizontes en la creación de programas, siguen siendo bastante limitados en comparación a lenguajes que requieren pura programación.

Su soporte para la programación orientada a objetos(P.O.O) resulta bastante complejo por lo que no se recomienda este lenguaje para este tipo de formas de programar.
