Ciclos o Bucles



Podemos escribir un procedimiento de Visual Basic que permita que el programa se ejecute repetidamente hasta que se cumpla una condición o un conjunto de condiciones. Este procedimiento se conoce como looping. El bucle es una característica muy útil de Visual Basic porque facilita los trabajos repetitivos. Hay tres tipos de bucles en Visual Basic, el bucle Do Loop , el bucle For Next y el bucle While Wend.
